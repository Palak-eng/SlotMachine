# 🎰 Unity Slot Machine Game

A simple and polished slot machine game developed in Unity using C#.
This project was created as part of a Unity game development assignment.

---

# 📌 Features

* Smooth reel spinning animation
* Randomized slot outcomes using RNG
* Win detection based on matching symbols
* Multiple betting options (10G, 50G, 100G)
* Dynamic reward system based on selected bet
* Balance management system
* Responsive UI interactions
* Sequential reel stopping for improved game feel
* WebGL build support

---

# 🛠️ Technologies Used

* Unity
* C#
* TextMeshPro
* Unity UI System

---

# 🎮 Gameplay Rules

* Player selects a bet amount.
* Press the SPIN button to start the slot machine.
* If all three center symbols match, the player wins.
* Reward amount depends on:

  * Selected bet value
  * Winning symbol type

---

# 💰 Betting System

| Bet Amount | Reward Range   |
| ---------- | -------------- |
| 10G        | Lower rewards  |
| 50G        | Medium rewards |
| 100G       | Higher rewards |

Higher bets provide higher potential rewards.
